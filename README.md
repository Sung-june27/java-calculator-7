# java-calculator-precourse

## ✅ 기능 목록

-[ ] **구분자를 기준으로 문자열 분리**
    -[ ] 커스텀 구분자 유무 확인
    -[ ] 쉼표(,) 또는 콜론(:)
    -[ ] 커스텀 구분자(문자열 앞부분의 "//"와 "\n" 사이에 위치하는 문자)
-[ ] **구분된 문자열을 숫자로 변환**
-[ ] **각 숫자의 합 반환**
-[ ] **입력값 검증**
    -[ ] 입력값이 양수인지 검증
    -[ ] 잘못된 값이 입력된 경우, IllegalArgumentException 발생 후 애플리케이션 종료
